# **App Name**: TON Gift Marketplace

## Core Features:

- Profile Display: Show user profile details including username (e.g., Azico_uzb), volume, bought/sold stats, cashback bonus, season achievements, giveaways, portals level, referrals and friends volume, following the design of the provided image of Azico_uzb profile
- Gift Inventory: Enable users to view, manage, and list their NFT Gifts, with options to filter by listing status (Unlisted/Listed).  Allow to add/mint new NFT, withdraw NFT, Sell NFT, and Send to another user.
- NFT Marketplace: Browse NFTs listed by other users, filter by price, rarity, and gift type; search by name or ID; display trending gifts, new listings, and featured collections
- Friend Invitation: Generate referral links. Track friend's volume for cashback.
- Telegram Integration: Set up bot with /start command using bot.py (similar to node.js bot). Use this as a tool for general application onboarding.
- Search and Filter: Allow users to filter the NFT based on different parameters. NFT can be filtered based on Collections, model and background.
- Telegram Bot Onboarding: When a user presses start on the bot, send a welcome message: 'Welcome! Find, trade and collect unique digital gifts in our marketplace. Start discovering now!' with two buttons: 'Open Web App!' and 'Join Community!'. The second button should link to a Telegram channel. The first button activates the node.js code.
- NFT Store: Display a list of NFTs available for purchase, with filtering and sorting options.

## Style Guidelines:

- Dark background with blue (#29ABE2) and green (#90EE90) accents, inspired by the NFT aesthetic.
- Primary color: A vibrant, desaturated cyan (#73E8DE) evoking a modern and digital feel.
- Background color: A very dark gray (#121212), creating a sleek and immersive environment that emphasizes the bright UI elements.
- Accent color: A vivid lime green (#BFFF00), contrasting against the primary and background for interactive elements and key information.
- Headline font: 'Poppins', a geometric sans-serif for a modern and fashionable look.
- Body font: 'Inter', a grotesque-style sans-serif providing a clean, neutral look, and readability, especially in a dark mode environment.
- Custom designed icons for profile sections, NFT types, and actions, with a modern, minimalist style, in a lighter shade of green.
- Clean and intuitive layout, emphasizing key information and easy navigation, similar to the reference image of Azico_uzb's profile.
- Subtle animations for NFT transitions, loading states, and user interactions, enhancing the overall user experience.
